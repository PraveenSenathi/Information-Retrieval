Implemented heuristics for sentimental analysis in Information retrieval     	 
Developed an application to deduce the polarity of all the synsets in the wordnet2.0 dictionary

Used the Stanford Part of speech tagger,stemmer and third party Java API JAWS for extracting information from the dictionary and implementing the rules to get the polarities
